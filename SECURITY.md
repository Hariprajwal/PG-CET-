# Security Policy

## Supported Versions

Currently, this repository strictly contains static study materials and PDFs. There are no active software versions being maintained. 

| Version | Supported          |
| ------- | ------------------ |
| Main    | :white_check_mark: |

## Reporting a Vulnerability

If you discover any security issues with the repository or its contents (such as malicious files masquerading as PDFs), please report it immediately.

You can report vulnerabilities by opening an issue on this repository or contacting the maintainer directly. We will try to address the issue as soon as possible and remove any harmful files.
