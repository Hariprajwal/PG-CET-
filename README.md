This repository serves as a centralized hub for **Karnataka PGCET** preparation. It contains a curated set of previous year question papers and related documents essential for cracking the exam.

### ✨ Features
- **Historical Papers:** Access PGCET papers dating back to 2009.
- **Instant Access:** PDF format for easy download and offline viewing.
- **Open Source:** Feel free to contribute or use the resources for your preparation.

## 📂 Contents

- 2009 to 2020 Previous Year Question Papers.
- 2025 Computer Science Engineering Question Paper with Solutions.
